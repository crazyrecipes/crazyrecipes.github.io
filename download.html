Run RecipeBuddy locally

Get the full functionality of RecipeBuddy in-house. Simply download and run the .jar and navigate to http://localhost:8080 in your browser of choice (we recommend Chrome or Firefox).

Build RecipeBuddy yourself

If you're feeling bolder, you can build RecipeBuddy directly from the source code. You will need the Java 17 JDK and the Intellij IDEA IDE to open the project proprely.
